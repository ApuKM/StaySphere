# 🏡 StaySphere

StaySphere is a modern, full-stack property and accommodation listing platform built with **Next.js (App Router)**, **MongoDB**, and **HeroUI**. It allows users to browse properties, filter listings, manage profiles, and securely add or edit their own property listings.

## ✨ Features

- **🔒 Secure Authentication:** Edge-compatible middleware for protecting private routes (e.g., `/items/add`, `/my-profile`).
- **🏠 Dynamic Listings:** Browse properties with real-time URL-based filtering (Location, Property Type).
- **📄 Advanced Pagination:** Server-side pagination synchronized with the UI using HeroUI components.
- **👤 User Profiles:** Beautiful, responsive profile management system.
- **🎨 Modern UI:** Built with **Tailwind CSS** and **HeroUI** for a fully responsive, accessible, and premium look.
- **⚡ Server-Side Rendering (SSR):** Optimized performance and SEO using Next.js Server Components and Route Handlers.

---

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router, Server Components, Server Actions)
- **Language:** TypeScript
- **Database:** [MongoDB](https://www.mongodb.com/) (Native Node.js Driver)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Components:** [HeroUI (@heroui/react)](https://heroui.com/)
- **Authentication:** [Better Auth / NextAuth] (Edge-compatible session management)
- **Icons:** React Icons
